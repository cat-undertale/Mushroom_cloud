# mushrooms_platform
毕设，基于微服务的食用菌种植与交流平台
项目采用前后端分离开发

## 3.1后端技术
- springboot
- springcloud
- MybatisPlus
- spring security
- redis
- maven
- easyExcel
- jwt
- OAuth2
## 3.2前端技术
vue + element-ui + axios + node.js…

## 3.3其他技术
- 阿里云oss(存储服务,用来存东西)
- 阿里云视频点播服务
- 阿里云短信服务
- 微信支付和登录
- docker
- git
- jenkins
